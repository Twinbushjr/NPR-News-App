# NPR-News-App
Demo:  https://npr-app.herokuapp.com/

App scrapes news articles from npr.org and allows users to save and leave comments about those articles.
Users May also delete comments if they please.

NPM Packages used:

<ul>
  <li>Express</li>
  <li>Axios</li>
  <li>Request</li>
  <li>Body-Parser</li>
  <li>Cheerio</li>
</ul>
